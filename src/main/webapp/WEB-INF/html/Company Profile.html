Company profile / 公司简介
  • 北京海科融通支付服务股份有限公司
    成立于2001年4月，注册资本2.558亿元，是从事金融电子支付和信息安全领域相关产品研发、生产、销售的综合性高新技术企业，于2011年获得中国人民银行颁发的全国范围内银行卡收单业务类型《支付业务许可证》，拥有全国范围内经营第三方支付业务的从业资质，并于2016年12月20日续牌成功，牌照有效期延至2021年12月21日。公司总部设在北京，在全国共设有33家分公司，与中国银联、中国银行、招商银行等30多家金融机构达成了稳固的合作关系。
  • 2018年，北京海科融通支付服务股份有限公司与深圳市中付天下电子支付有限公司达成战略合作，开启支付通2.0新模式，米仓伙伴就此诞生
  • 深圳市中付天下电子支付有限公司
    深圳市中付天下电子支付有限公司是经有关部门支持下从事金融外包服务公司，成为国家认证银联POS机收款机和线上、线下支付终端销售为一体的供应服务商。专业为各类企事业单位、中小企业、连锁机构、个体工商户提供终端解决方案和终端支付结算平台的服务机构。凭借长期对支付行业和金融外包服务的业务积累， 紧密围绕持卡人、商户、收单行利益，以我们专业的服务团队和第三方支付单位提供的高效的数据处理中心；并依托银行间跨行交换和清算网络，利用先进的通讯技术，为各银行、 商户、持卡人提供专业、稳定、 高效的银行卡线上、线下支付服务。有最具实力的技术团队和最强大的第三方支持。

    公司现已在深圳横岗、深圳沙井、长沙、安徽、柳州、南宁成立分公司，全国代理500多家，加入我们，共享财富！！
